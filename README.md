# YouTube Reklama Skip

Расширение для браузера, которое позволяет автоматически (или вручную) пропускать спонсорские вставки, интро и интеграции на YouTube при помощи ИИ Gemini.

## Стек технологий
- Vanilla JavaScript
- Vanilla CSS (с поддержкой CSS переменных и сучасных blur/glassmorphism эффектов для popup)
- Chrome Extension Manifest V3

## Как запустить
1. Склонируйте репозиторий.
2. Откройте `chrome://extensions/` или `edge://extensions/` в вашем браузере.
3. Включите режим разработчика (Developer mode).
4. Нажмите "Загрузить распакованное расширение" (Load unpacked) и выберите папку с этим проектом.

## Особенности
- **Фоновый анализ**: Расширение автоматически открывает вкладку Gemini и просит проанализировать текущее видео.
- **Точные тайминги**: Полученные тайминги отрисовываются прямо на прогресс-баре плеера YouTube.
- **Минимальная нагрузка на ПК**: Никаких тяжелых фреймворков. Написано на чистом JS.
- **Локальный кэш**: Тайминги сохраняются в `chrome.storage.local` для мгновенного доступа при повторном просмотре.
